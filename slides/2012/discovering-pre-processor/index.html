C'est quoi un pré processeur CSS ?

Un pré-processeur CSS... Ben, c'est comme si tu mets des petites roues à ton vélo, sauf que quand tu les mets, ben ça transforme ton vélo en Poney.
Donc du coup, tes stylesheets elles roxent grâve sa maman.

- Maxime Thirouin / @MoOx

Les maths

1em - 1em;    // 0em
1in + 72pt;   // 2in
6px * 4;      // 24px
18  % 5;      // 3

Division, cas spécial

font   : 18px / 1.45em; // 18px/1.45em
font   : (20px / 5);    // 4px
font   : 20px / 5 + 1;  // 5px
font   : $base / 5;     // 4px
$size  : 20px / 5;      // 4px

Conditions

Relational operators (<, >, <=, >=)

1  <  20  // true
10 <= 20  // true
4  >  1   // true
4  >= 1   // true

Comparison operators (==, !=)

1 + 1 == 2     // true
small != big   // true
#000  == black // true

Conditions directives

@if, @else

$theme: ocean;
div {
@if $theme == dusty {
background: #c6bba9;
color: $color;
} @else if $theme == ocean {
background: blue;
color: white;
}

box-shadow: 0 2px 0 if($theme == ocean, #c6bba9, #000); // if() function !
}

mais aussi @while, @each, @for

Les MIXINs

Produisent de la sortie CSS 
@mixin position($position, $top: null, $right: null, $bottom: null, $left: null)
{
position: $position;
top: $top;
right: $right;
bottom: $bottom;
left: $left;
}

.fixed-header {
@include position(absolute, 0, $left: 0, $right: 0);
}
ex: gradient - tooltip

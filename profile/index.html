Maxime Thirouin,

a front-end developer who loves his job

Front-End

I like to work on front-end, whether it's for websites or web applications, using web technologies availables, including HTML5, CSS 3 (with CSS pre-processor like Sass) and JavaScript.

Whenever I develop, I also pay big attention to adaptive web design and web performances, to serve accessible content for each devices, regardless of the size and the bandwidth available. I take care about accessibility to everyone, including search engines.

Back-End

I came from a software training, and used to work a lot on back-end using PHP (sometimes with the Zend Framework or Symfony), and SQL (MySQL or Postgres).
Now I prefer Node.js and use it even for front-end stuff (as a server, or mostly to handle commons task with Grunt).
I'm already code with Ruby & Python, but I've to admit that I'm still a noob with these languages.
